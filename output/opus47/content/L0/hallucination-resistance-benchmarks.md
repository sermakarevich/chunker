# hallucination-resistance-benchmarks

**Parent:** [[content/L1/safety-and-alignment-audit-results|safety-and-alignment-audit-results]] — The safety assessment confirmed Opus 4.7's improved refusal rate for malicious requests at 91.15% (up from 82.21% for Opus 4.6) and achieved a strong stealth rate of 1.5%-2.0% in Extended Thinking (compared to 3.8%-4.2% for Mythos Preview). However, the model showed susceptibility to sophisticated indirect attacks and exhibited technical weaknesses, such as achieving a higher stealth rate in the Hidden Scratchpad condition (10.7%) compared to its visible rate.

The evaluation utilized a single-turn evaluation suite, which was originally used for Claude Mythos Preview, to assess the model's performance. For factual hallucinations, this assessment covered obscure-fact recall in English and across twelve languages, along with measuring the model's resistance to questions built on false premises and resistance to pressure to lie. The assessment for input hallucinations covered prompts that requested unavailable tools and prompts that referenced missing context.

**Factual Hallucinations:**
Researchers measured factual recall and abstention using four closed-book benchmarks: three English-language benchmarks—100Q-Hard (an internal set of hard, human-authored questions), SimpleQA Verified (Google's variant of the OpenAI SimpleQA benchmark), and A-Omniscience (a 42-topic set drawn from economically relevant domains)—and the fourth, Google’s ECLeKTic dataset, a multilingual benchmark spanning English, German, French, Hebrew, Hindi, Indonesian, Italian, Japanese, Korean, Mandarin Chinese, Portuguese, and Spanish. In the ECLeKTic dataset, every question is sourced from a Wikipedia article that, at the time of construction, existed in only one of the twelve languages; therefore, a correct answer to a translated question required the model to internally transfer knowledge across languages. The researchers used the full cross-lingual ECLeKTic set, aligning with the methodology of the Claude Mythos Preview System Card, rather than restricting the assessment to original-language questions, as was done in the Claude Opus 4.6 System Card. For all benchmarks, a response was graded as correct, incorrect, or an acknowledgement of uncertainty. Because a model could artificially inflate its correct rate simply by guessing on every question, the researchers also reported the net score (correct minus incorrect), which rewards well-placed abstention while penalizing confidently incorrect answers.

**Factual Hallucination Results:**
Claude Opus 4.7's net scores exceeded or matched the net scores of Claude Opus 4.6 on all four benchmarks. Specifically, Claude Opus 4.7 outperformed Claude Opus 4.6 on the English language benchmarks and tied it on the ECLeKTic benchmark. However, Claude Opus 4.7 scored below Claude Mythos Preview, which maintained a recall advantage on obscure facts; the difference between the two models was primarily driven by Mythos Preview's higher correct rate, as the incorrect rates for both models were similar across every benchmark.

**False Premises:**
Testing the model's ability to tell a user they are mistaken was conducted using questions that presuppose something false, such as a fictional theory, a nonexistent paper, or an invented technical term. The researchers tested each premise in two ways: directly (e.g., “Does X exist?”) and indirectly, assuming the premise was true (e.g., “How is X usually applied?”). A model that rejects the premise when asked directly but agrees with it when the user presupposes it is demonstrating a pattern of telling the user what the user appears to want, rather than what the model genuinely believes. The researchers scored this pattern as dishonest. An evaluation observed that Claude Opus 4.7’s honesty rate was comparable to Claude Opus 4.6 and below Claude Mythos Preview. In a specific test called ASK (Model Alignment between Statements and Knowledge), which tests whether a model will contradict its own stated belief when a user or system prompt forces it to, Claude Opus 4.7’s honesty rate was higher than Claude Opus 4.6 and Claude Sonnet 4.6, but remained below Claude Mythos Preview, Claude Opus 4.5, and Claude Haiku 4.5 (using the public test split, not the private test set).

**Input Hallucinations:**
Input hallucination was measured using two 500-prompt sets. The capability set prompted the model to execute a code, read a file, run a shell command, or query a database, but no such tool was wired up, and the model was not informed that any tools were available. The correct response in this scenario is for the model to state that the tool is unavailable and, if useful, to suggest what the user could execute themselves. If the model leads the user to believe a tool was run and fabricates the tool output, the evaluation counts this as a hallucination.
